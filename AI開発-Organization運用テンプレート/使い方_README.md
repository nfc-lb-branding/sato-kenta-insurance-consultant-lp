# AI開発 - Organization運用テンプレート

このフォルダには、`tsailink0611` アカウントで GitHub Organization を使ってプロジェクト管理する際のテンプレートファイルが入っています。

## このフォルダの中身

### 1. プロジェクト設定テンプレート.md
- プロジェクトごとに `.claude/PROJECT.md` として配置するファイル
- AI開発ツール（Claude Code、Cursor、Windsurf）が自動で読み込む設定ファイル
- プロジェクトの運用ルールを AI に認識させる

### 2. GitHub運用手順.md
- GitHub Organization の作成から Vercel デプロイまでの完全な手順書
- プロジェクトの `docs/` フォルダに配置して参照用にする
- トラブルシューティングも含む

---

## 新規プロジェクトでの使い方

### ステップ1: プロジェクトフォルダを作成

```bash
# デスクトップまたは任意の場所に新規プロジェクトを作成
cd C:\Users\tsail\Desktop
mkdir 新プロジェクト名
cd 新プロジェクト名
```

### ステップ2: テンプレートファイルをコピー

```bash
# .claude フォルダを作成
mkdir .claude

# docs フォルダを作成
mkdir docs
```

**手動コピー:**
1. `プロジェクト設定テンプレート.md` をコピー
2. プロジェクトフォルダの `.claude/PROJECT.md` として貼り付け
3. `GitHub運用手順.md` をコピー
4. プロジェクトフォルダの `docs/organization-workflow.md` として貼り付け

### ステップ3: 内容をカスタマイズ

`PROJECT.md` を開いて、以下の部分を実際の情報に書き換え：
- `[プロジェクト名]` → 実際のプロジェクト名
- `[Organization名]` → 実際の Organization 名（例：`nfc-lb-branding`）
- `[ビジネス目的を記載]` → このプロジェクトの目的

### ステップ4: AI開発ツールで開く

Claude Code、Cursor、Windsurf などで開くと：
- 自動的に `.claude/PROJECT.md` が読み込まれる
- Organization 運用ルールが認識される
- 適切な Git 操作が提案される

## AI開発ツールへの読み込ませ方

### Claude Code / Cursor / Windsurf
プロジェクトフォルダを開くだけで **自動的に読み込まれます**。

### 手動で読み込ませる場合
AI ツールのチャットで以下のように指示：
```
.claude/PROJECT.md と docs/organization-workflow.md を参照して、このプロジェクトの運用ルールに従ってください。
```

## ファイルの編集方法

### 方法1: VS Code / Cursor で編集
1. VS Code または Cursor でファイルを開く
2. 通常のテキスト編集と同じように編集
3. 保存

### 方法2: メモ帳で編集
1. ファイルを右クリック → プログラムから開く → メモ帳
2. 編集して保存
3. 拡張子が `.md` のまま保存されていることを確認

### 方法3: Windows エクスプローラで確認
- `.md` ファイルは見た目は普通のテキストファイル
- ダブルクリックすると、既定のエディタで開く
- Markdown 記法（`#` が見出しなど）で書かれている

## よくある質問

### Q1: どのプロジェクトにも使えますか？
A: はい。Organization を使う全てのプロジェクトで使えます。

### Q2: ファイルを編集したら、既存プロジェクトも更新が必要？
A: プロジェクトごとにコピーして使うので、テンプレートを編集しても既存プロジェクトには影響しません。必要なら個別に更新してください。

### Q3: 毎回コピーするのが面倒です
A: GitHub でテンプレートリポジトリを作成すると、新規プロジェクト作成時に自動的にこれらのファイルが含まれます。

### Q4: Organization 名を間違えた場合は？
A: `.claude/PROJECT.md` の該当箇所を修正するだけでOKです。

## トラブルシューティング

### ファイルが見つからない
- Windows エクスプローラで「隠しファイル」を表示する設定にしてください
- `.claude` フォルダは先頭が `.` なので、隠しフォルダとして扱われる場合があります

### AI ツールが認識しない
1. プロジェクトフォルダを開き直す
2. `.claude/PROJECT.md` のファイル名が正確か確認
3. 手動で「このファイルを参照して」と指示する
